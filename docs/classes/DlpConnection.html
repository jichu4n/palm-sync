      • OptionalignoreErrorCode?: boolean | DlpRespErrorCode | DlpRespErrorCode[]

        Whether to throw an error when the response has a non-zero error code.

        By default, execute() will throw an error when the response has a non-zero error code.

        If ignoreErrorCode is set to true, execute() will ignore non-zero error codes and return the response as-is.

        If ignoreErrorCode is set to one or more error codes, execute() will ignore those specific error codes but will still throw an error for other error codes.
